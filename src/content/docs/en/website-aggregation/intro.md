## Overview

The AI Website Aggregation feature allows you to access multiple AI chat websites simultaneously in one interface, such as ChatGPT, Claude, Gemini, etc.

## Supported Platforms

- OpenAI (ChatGPT)
- Anthropic (Claude)
- Google (Gemini)
- Microsoft (Copilot)
