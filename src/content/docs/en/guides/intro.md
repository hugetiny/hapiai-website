## What is HapiAI?

HapiAI is a next-generation AI aggregation chat platform based on Perplexica. We are dedicated to solving the fragmentation of current AI tools, providing you with a one-stop AI interaction experience.

## Core Features

### 1. AI Website Aggregation Chat (Original)

Are you tired of switching back and forth between multiple websites like ChatGPT, Claude, Gemini, etc.? HapiAI's original aggregation chat feature allows you to access and manage these AI websites in a unified interface.

- **Unified Interface**: Say goodbye to browser tab chaos.
- **Efficient Switching**: Switch between different AI services with one click.

### 2. AI API Aggregation Chat

Based on the powerful Perplexica engine, we have integrated various mainstream AI APIs.

- **Multi-model Support**: Supports GPT-4, Claude 3, Llama 3, and more.
- **Web Search**: Built-in web search capability provides real-time and accurate answers.
