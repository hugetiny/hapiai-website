## Desktop Installation

HapiAI Desktop is currently in Alpha testing. You can download the latest version from our GitHub Release page.

### Windows

Download the `.exe` installer and run it.

### macOS

Download the `.dmg` image file and drag the app into the Applications folder.

### Linux

`.AppImage` and `.deb` packages are provided.

## FAQ

### When will the mobile version be released?

We are fully focused on developing the desktop version. Mobile development is planned to start after the desktop version stabilizes.
